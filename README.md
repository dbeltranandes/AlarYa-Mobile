#### Notas adicionales
1. La aplicación se desarrolló en Kotlin
2. En el repositorio hay dos videos (de 1m aprox c/u) del funcionamiento de la app para su validación: Una con el emulador y otra con la apk instalada en el celular
3. En coursera se anexó la apk pero de igual manera aquí se encuentra el mismo archivo en la raíz del proyecto
4. Para validar el proyecto únicamente se debe clonar la aplicación y correr con Android Studio, para mejor entendimiento, división de tareas y revisión se dividó cada pantalla en un archivo .kt para cada una de ellas. Si se desea revisar el código de una pantalla en particular solo sería dirigirse a la pantalla que hace referencia
5. Todos los cambios se encuentran en esta rama, algunas pantallas se realizaron por medio de ramas adicionales y se integraron a la principal
